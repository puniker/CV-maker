# CV Maker

Aplicación con react y Firebase para genererar CV personalizados.

## Levantar entorno de local

```bash
npm install
```

```bash
npm run dev
```

## Deploy en entorno de desarrollo

Abrir PR a la rama release/develop y la pipelane se ejecutará automaticamente.
